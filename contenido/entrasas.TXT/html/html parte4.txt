<b>ETIQUETA CON FORMATO:</b><br />
Algunas etiquetas de HTML5 nos devuelven un formato especifico entre ellos tenemos el</div>
<div>
formato<b>&nbsp;</b>NEGRITA, CURSIVA, TACHADO, SUBRAYADO, SUPERÍNDICE, entre otros. Pero ojo HTML no es un lenguaje de estilos.<br />
<br />
<br />
<br />
Vamos a conocer algunas de estas etiquetas.<br />
<br />
<div class="separator" style="clear: both; text-align: center;">

<br /></div>
<div>
<span style="color: #38761d;"><b>&lt;b&gt;&lt;/b&gt; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</b></span><span style="color: #38761d;"><span style="color: #666666;"><b>----&gt;</b></span><b style="color: #666666;"> </b>Es una etiqueta que devuelve texto con <b>formato negrita.</b></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;i&gt;&lt;/i&gt; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span><span style="color: #38761d;">----&gt; Es una etiqueta que devuelve texto con <i><b>formato cursiva</b>.</i></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;mark&gt;&lt;/mark&gt;&nbsp;</span><span style="color: #38761d;">----&gt; Es una etiqueta que devuelve <b style="background-color: yellow;">texto con formato marcado o resaltado.</b></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;del&gt;&lt;/del&gt; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</span><span style="color: #38761d;">----&gt; Es una etiqueta que devuelve texto con<b> <strike>formato tachado o eliminado</strike>.</b></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;ins&gt;&lt;/ins&gt; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;&nbsp;</span><span style="color: #38761d;">----&gt; Es una etiqueta que devuelve texto con<b> <u>formato subrayado</u>.</b></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;sub&gt;&lt;/sub&gt; &nbsp; &nbsp; &nbsp; &nbsp;</span><span style="color: #38761d;">----&gt; Devuelve texto con con base(YA LO VEREMOS EN EL EJEMPLO)</span><span style="color: #38761d;">.</span><br />
<b><span style="color: #38761d;">&lt;sup&gt;&lt;/sup&gt; &nbsp; &nbsp; &nbsp; &nbsp;</span><span style="color: #38761d;">----&gt;&nbsp;</span></b>Devuelve texto con con exponente(YA LO VEREMOS EN EL EJEMPLO).<br />
<span style="color: #38761d; font-weight: bold;">&lt;strong&gt;&lt;/strong&gt;&nbsp;</span><span style="color: #38761d;">----&gt; Es una etiqueta también&nbsp;devuelve<b> texto en negrita .</b></span><br />
<span style="color: #38761d; font-weight: bold;">&lt;em&gt;&lt;/em&gt; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;</span><span style="color: #38761d;">----&gt; Esta etiqueta hace referencia a<b>&nbsp;</b></span><b><i>énfasis</i></b></div>
<div>
<br /></div>
<div>
De todas estas etiquetas algunas ya no se usan, estas se dejaron de usar ya que ahora algunos de estos formatos que s ele da al los textos se le puede hacer con CSS.<br />
<br /></div>
<div>
<b>DIFERENCIA&nbsp;ENTRE ETIQUETA &lt;b&gt; Y &lt;strong&gt;</b></div>
<div>
<br />
Entre estas etiquetas no hay diferencia visual pero si hay diferencia semántica, la etiqueta &lt;b&gt; solo te devuelve un formato, la etiqueta &lt;strong&gt; significa un texto fuerte o un texto importante.<br />
<br />
<div>
<b>DIFERENCIA&nbsp;ENTRE ETIQUETA &lt;i&gt; Y &lt;em&gt;</b></div>
<div>
<br />
Entre estas etiquetas visualmente también te devuelven lo mismo pero sistemáticamente no son lo mismo, la etiqueta &lt;i&gt; solo te devuelve un texto con formato cursiva, mientras que la etiqueta &lt;em&gt; un texto en énfasis un texto que sobresale de los demás.<br />
<br />
Para aprovechar este tema también aprenderemos las etiquetas de Listas.<br />
<br />
Existen dos tipos de listas en HTML5 y para añadir listas en html se usan 3 etiquetas una etiqueta para cada tipo de lista y la otra es para definir los elementos dentro de la lista .<br />
<br />
LISTAS ORDENADAS: Se caracterizan por llevar números sucesivos delante de cada elemento para ello se usa la etiqueta &lt;ol&gt;&lt;/ol&gt;<br />
<br />
LISTAS DESORDENADAS: estas solo llevan una viñeta delante de cada elemento, para esta lista se usa la etiqueta &lt;ul&gt;.&lt;/ul&gt;<br />
